# Minions Blogs

Blog definitions, settings, brand voice, and target audience profiles

## Overview

This project is part of the [Minions ecosystem](https://github.com/mxn2020/minions) and provides:

- **TypeScript SDK** (`@minions-blogs/sdk`) — Core library for Node.js
- **Python SDK** (`minions-blogs`) — Python bindings
- **CLI Tool** (`@minions-blogs/cli`) — Command-line interface
- **Web App** — Interactive playground at [blogs.minions.wtf](https://blogs.minions.wtf)
