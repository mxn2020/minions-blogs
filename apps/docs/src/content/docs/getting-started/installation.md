---
title: Installation
description: Install Minions Blogs packages
---

## TypeScript / Node.js

```bash
npm install @minions-blogs/sdk minions-sdk
```

## Python

```bash
pip install minions-blogs
```

## CLI

```bash
npm install -g @minions-blogs/cli
```

## Requirements

- **Node.js** 20 or later
- **Python** 3.11 or later (for Python SDK)
