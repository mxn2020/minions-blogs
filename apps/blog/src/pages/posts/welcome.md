# Welcome to Minions Blogs

We're excited to introduce **Minions Blogs** — Blog definitions, settings, brand voice, and target audience profiles.

## What's Included

- **TypeScript SDK**: `npm install @minions-blogs/sdk`
- **Python SDK**: `pip install minions-blogs`
- **CLI**: `npm install -g @minions-blogs/cli`

## Getting Started

Check out the [documentation](https://blogs.minions.help) for a full guide.
